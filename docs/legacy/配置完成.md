## 🚀 使用指南

### 1. 索引文件

```bash
# 索引默认目录 (~/Documents, ~/Desktop, ~/Downloads)
python3 main.py index
```

首次运行会扫描并索引所有支持的文件类型。

### 2. 搜索文件

```bash
# 语义搜索文件
python3 main.py search "预算报告"

# 搜索幻灯片
python3 main.py search "产品规划" --type slides --limit 20

# 关键词搜索
python3 main.py search "技术文档" --limit 15
```

### 3. 查看统计

```bash
python3 main.py stats
```

显示索引文件数、token 使用量、成本等信息。

## 🔧 故障排查

### 问题：索引速度慢

**解决方案**：
- 减少扫描目录
- 使用更小的嵌入模型 (bge-small)
- 增加 `MAX_CONCURRENT_LLM` 配置

### 问题：内存占用高

**解决方案**：
- 使用 bge-small 而不是 bge-large
- 减少并发数
- 限制文件大小 `MAX_FILE_SIZE_MB`

### 问题：搜索结果不准确

**解决方案**：
- 使用更大的嵌入模型 (bge-base 或 bge-large)
- 重新索引文件
- 使用更具体的搜索词
