# 智能文件检索工具 - 快速开始

## 项目概述

这是一个基于 AI 的智能文件检索工具，可以深度理解和搜索您的本地文件，包括文档、演示文稿、图片、代码等。

## 已实现功能 (Phase 1 MVP)

✅ **文件扫描与解析**
- 支持多种格式：PDF, Word, PowerPoint, 图片, 文本, 代码等
- 智能类型检测和内容提取
- 幻灯片级别的解析（PPT/PPTX）

✅ **AI 驱动的内容理解**
- 使用 Claude 3.5 进行内容摘要和分析
- 自动提取关键词和分类
- 图片内容识别（Vision API）

✅ **语义搜索**
- 基于向量嵌入的语义搜索
- 文件级和幻灯片级搜索
- 相似度排序

✅ **成本控制**
- 每日/每月预算限制
- Token 使用追踪
- 成本统计

✅ **隐私保护**
- 本地数据存储
- 敏感目录黑名单
- 可配置的隐私模式

## 快速开始

### 1. 安装依赖

```bash
# 运行安装脚本
./setup.sh

# 或手动安装
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. 配置

编辑 `.env` 文件，添加 API 密钥：

```bash
ANTHROPIC_API_KEY=your_key_here
OPENAI_API_KEY=your_key_here
```

配置扫描目录（可选）：
```bash
SCAN_DIRECTORIES=~/Documents,~/Desktop,~/Downloads
```

### 3. 初始化数据库

```bash
python main.py init
```

### 4. 索引文件

```bash
python main.py index
```

这将扫描配置的目录并索引所有支持的文件。

### 5. 搜索

搜索文件：
```bash
python main.py search "预算报告"
```

搜索幻灯片：
```bash
python main.py search "竞品分析" --type slides --limit 20
```

### 6. 查看统计

```bash
python main.py stats
```

## 技术栈

- **语言**: Python 3.8+
- **数据库**: SQLite + 向量存储
- **LLM**: Claude 3.5 Sonnet/Haiku
- **嵌入**: OpenAI text-embedding-3-small
- **解析**: PyPDF2, python-docx, python-pptx, Pillow

## 常见问题

### 1. 如何控制成本？

在 `.env` 中设置预算：
```bash
DAILY_BUDGET_USD=10.0
MONTHLY_BUDGET_USD=100.0
```

### 2. 如何排除敏感目录？

在 `.env` 中配置黑名单：
```bash
BLACKLIST_DIRS=.ssh,Library/Keychains,Private
```

### 3. 支持哪些文件格式？

- 文档: PDF, DOCX, TXT, MD, HTML
- 演示: PPTX (Keynote 和 WPS 将在 Phase 3 支持)
- 图片: JPG, PNG, GIF, WebP, TIFF
- 代码: Python, JavaScript, Java, C/C++ 等
- 其他: CSV, XLSX 等

### 4. 搜索速度如何？

- 文件级搜索: < 1 秒（1万文件）
- 幻灯片级搜索: < 2 秒（10万页）

### 5. 数据存储在哪里？

所有数据本地存储在 `data/` 目录：
- 数据库: `data/db/files.db`
- 缓存: `data/cache/`
- 缩略图: `data/thumbnails/`
