# Demo Assets

Place `demo.gif` in this directory. Both README.md and README_zh.md reference it.

## Recording the demo

Record a 10-15 second screen capture showing:

1. User types a natural language query (e.g., "find my Q4 budget report")
2. OpenFiles returns matching files with similarity scores
3. User asks a follow-up question
4. AI responds with an answer citing specific files

Recommended tools: [Kap](https://getkap.co/) (macOS), [Peek](https://github.com/phw/peek) (Linux), or [ScreenToGif](https://www.screentogif.com/) (Windows).

Recommended size: 720px wide, keep file under 5MB for fast GitHub loading.
